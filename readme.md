             GymPro 終極智慧管理系統 - 專案說明文件 (README)
========================================================================

一、系統簡介
------------------------------------------------------------------------
本系統為一款基於 Java Swing 與 MySQL 開發的視窗應用程式，旨在提供全方位的
健身房管理解決方案。系統整合了資料視覺化 (JFreeChart) 與報表匯出 (Apache POI) 
功能，並分為「一般會員」與「管理員」兩種權限介面。


二、系統需求 (System Requirements)
------------------------------------------------------------------------
1. 作業系統：Windows 10/11, macOS, 或 Linux
2. Java 版本：Java 11 或以上 (建議使用 Java 17 以獲得最佳效能)
3. 資料庫  ：MySQL 8.0
4. 必要檔案：
   - gym-system.jar (主程式)
   - gym_system.sql (資料庫還原檔 - 包含結構與資料)
   - src 資料夾 (資源檔，內含圖片)


三、安裝與執行步驟 (Installation)
------------------------------------------------------------------------

[步驟 1] 資料庫設定
1. 請開啟 MySQL Workbench。
2. 進入 Server -> Data Import。
3. 選擇 "Import from Self-Contained File"，並選取專案附帶的 `gym_system.sql`。
4. 點擊 "Start Import"。
   (備註：SQL 檔已包含 Create Schema 指令，匯入後會自動建立 gym_system 資料庫)

[步驟 2] 檔案位置配置 (★非常重要★)
為了確保課程圖片能正確載入，請務必依照以下結構放置檔案：

您的專案資料夾/
  ├── gym-system.jar
  └── src/
       └── main/
            └── resources/
                 └── img/  <-- 這裡面必須包含 .jpg 圖片檔案

[步驟 3] 啟動系統
1. 開啟命令提示字元 (CMD) 或終端機。
2. 使用 `cd` 指令切換到 `gym-system.jar` 所在的資料夾。
   (例如：cd C:\Users\User\Desktop\GymProject)
3. 輸入以下指令並按下 Enter：
   
   java -jar gym-system.jar

五、系統功能亮點 (Features)
------------------------------------------------------------------------

[一般使用者功能]
1. 選購課程：
   - 瀏覽課程列表（含圖片預覽）。
   - 真實顯示「剩餘名額」(總名額 - 已報名人數)。
   - 自動檢查餘額與衝堂機制。

2. 我的課表：
   - 查看已選課程詳情。
   - [退選功能]：退選後，名額自動釋出 (+1)，費用自動退還。
   - [匯出 Excel]：產出包含樣式排版的專業課表 (.xlsx)。

3. 數據分析：
   - 透過圓餅圖 (Pie Chart) 視覺化檢視熱門課程分佈。

4. 個人中心：
   - 修改個人資料與線上儲值模擬。

[管理員後台功能] (需使用 admin 登入)
1. 課程管理：
   - 新增/刪除課程。
   - [資料回填]：點選表格列，資料自動帶入下方欄位，方便修改。
   - 檢視詳細報名數據。

2. 會員管理：
   - 檢視所有會員列表。
   - 修改會員權限 (如：升級為 ADMIN) 與餘額。
   - [強制刪除]：刪除會員時，系統會自動清除該會員的選課與登入紀錄，確保資料完整性。
